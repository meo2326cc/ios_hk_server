# 香港研究小組資料庫

使用 Node.js 來處理 google sheets API 相關功能的服務

**環境變數：**

`.env` 中需要有以下變數
- GOOGLE_SHEET_BOOK
- PORT
- API_KEY
- GOOGLE_SHEET_GOVP
- GOOGLE_SHEET_UKNA
- GOOGLE_SHEET_PERO
- GOOGLE_SHEET_NGOP
- GOOGLE_SHEET_COMM_PUBL
- GOOGLE_SHEET_ELEC
- GOOGLE_SHEET_COMM_POST
- GOOGLE_SHEET_OVSE
- GOOGLE_SHEET_NGOA
- GOOGLE_SHEET_POPY
- GOOGLE_SHEET_LEGC

**執行：**

```shell
# 安裝所需要的套件
npm i

# 啟動
npm start
```
